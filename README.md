# 电赛整理(2018.5-2019.8)

这个仓库是我们队伍在电赛集训期间的培训内容汇总

### 数字钟(2018.5)

主要涉及单片机的使用

### 小车(2018.7)

主要涉及步进/直流电机控制、简单传感器的使用、小车的任务控制

### 手势识别系统(2018.7)

主要涉及IIC通信、FDC2214传感器的使用、KNN算法等

### 万用表(2018.10-2018.12)

主要包括电容、电阻、电感的测量电路

### 简易旋转倒立摆(2019.1)

主要涉及串级PID的使用

### 无人机(2019.3-2019.8)

分为两个阶段，第一阶段3月-6月，对应文件夹`05四旋翼无人机`，期间主要完成无人机平台的搭建，包括姿态融合算法、飞控算法等；第二阶段为6月-8月，完成了两道无人机培训题目以及最终的电赛国赛，对应文件夹`06`、`07`、`08`
